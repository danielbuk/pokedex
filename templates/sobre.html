Sobre o Projeto

Esta Pokedex digital foi criada com muito carinho para todos os fãs de Pokémon! É um projeto que combina tecnologia moderna com a nostalgia dos jogos clássicos.

Tecnologias Utilizadas
  • Flask - Framework web Python
  • Bootstrap 5 - Framework CSS responsivo
  • JavaScript - Interatividade e animações
  • PokeAPI - Dados oficiais dos Pokémon
  • Font Awesome - Ícones modernos
Características
  • 100% Responsivo - Funciona em qualquer dispositivo
  • Dados Oficiais - Informações precisas da PokeAPI
  • Design Moderno - Interface intuitiva e bonita
  • Busca Inteligente - Encontre qualquer Pokémon rapidamente
  • Gratuito - Sem custos ou anúncios
